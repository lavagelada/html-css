paragrafos e quebras de linha

o paragrafo pode ser escrito de qualquer maneira, so precisa estar dentro do <p> e para fechar o paragrafo a gente usa o fim da tag </p>.

e para fazer uma quebra de linha
como essa (break row) eh so usar <br>.

e para essa quebra de linha com uma linha a tag tambem vazia eh <hr>, usado para destacar a quebra de linha para outro assunto

Vamos adicionar alguns simbolos, tambem conhecidos como HTML ENTITIES como:
marca registrada: ®
copy right: ©
trade mark: ™

Para fazer emojis vamos acessar o link do emojipedia e escolher qual emoji nos queremos: nos seguintes detalhes:
o codigo do emoji deve ser colado apos um "&#x" e so depois o codigo do emoji, acabando em ; . EX:

para escrever um 🖖, escrevi no codigo &#x + 1F596 + ;
e assim sucessivamente para outros emojis como
🙈, 😳, 👀 etc.
